# My Profile

## 専門分野

主に電気回路を触っています。VVVF インバータ（パワーエレクトロニクス）から省電力デバイスまで幅広く作りたいものを作ります

ハードウェア設計・組み込みソフトウェア開発・サーバ運用等趣味の範囲ながらいろいろやってます
