## 専門分野

主に電気回路を触っています。VVVFインバータ（パワーエレクトロニクス）から省電力デバイスまで幅広く作りたいものを作ります

ハードウェア設計・組み込みソフトウェア開発・サーバ運用等趣味の範囲ながらいろいろやってます
